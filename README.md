# goldfish4c

goldfish4c is a multiplatform Habbo Hotel V1 server implementation written in ANSI C.  
The project focuses on portability, simplicity, and adherence to early Habbo networking protocols, making it suitable for experimentation, research, and educational use.

## Overview

This repository provides a low-level C implementation of a classic Habbo Hotel server.  
By using ANSI C, the codebase avoids platform-specific dependencies and can be compiled with most standard C toolchains.
